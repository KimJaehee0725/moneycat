# Project Context

## Goal

- 커플 2인이 아이폰/PC에서 함께 사용하는 가계부 앱(MVP)을 안정화하고 점진적으로 UX 고도화.

## Current Decisions (summary)

- 신규 기능보다 실제 사용 피드백을 우선 수집하고 개선 항목을 배치 단위로 반영.
